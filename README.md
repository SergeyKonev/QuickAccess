# Quick Access

**Версия:** 1.3.0

Компактное расширение для быстрого перехода на подстраницы текущего сайта и автоматизации рутинных задач.

## Возможности

### Основные
- **Быстрый переход**: Удобная навигация по подстраницам текущего сайта (Quick Access).
- **Mantis Version Checker**: Скрипт для проверки версий в Mantis (`jabber.bx`).
- **Bitrix24 Integration**: Специфичные функции для работы с порталом.

### Функции Popup
Расширение предоставляет удобный popup-интерфейс со следующими возможностями:
- **Управление закладками**: Сохранение и быстрый доступ к важным ссылкам.
- **Сниппеты**: Управление текстовыми заготовками.
- **Kibana**: Интеграция для быстрого доступа к логам.
- **Controller Opener**: Инструмент для открытия контроллеров.
- **Тарифы**: Просмотр информации о тарифах.
- **Импорт/Экспорт**: Возможность резервного копирования настроек и закладок.

## Установка (для разработчиков)

1. Клонируйте репозиторий или скачайте исходный код.
2. Откройте браузер (Chrome или Edge) и перейдите по адресу `chrome://extensions/`.
3. Включите "Режим разработчика" (Developer mode) в правом верхнем углу.
4. Нажмите кнопку "Загрузить распакованное расширение" (Load unpacked).
5. Выберите папку с исходным кодом расширения.

## Структура проекта

- `background/` - Фоновые скрипты (Service Worker).
- `content-scripts/` - Скрипты, выполняемые на страницах веб-сайтов (например, Mantis).
- `popup/` - Исходный код всплывающего окна.
  - `controllers/` - Логика управления UI (модальные окна, импорт/экспорт).
  - `features/` - Реализация конкретных фич (сниппеты, кибана).
  - `services/` - Сервисные модули (работа с данными, сообщениями).
  - `views/` - Компоненты отображения.
- `shared/` - Общие утилиты и настройки, используемые в разных частях расширения.
